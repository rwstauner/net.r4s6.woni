# woni

Your data: Was Outside, Now Inside.


## Usage

### CLI

For usage info:

    $ clojure -M:run --help

or you can use `bin/cli ...`.
Run the cli passing in input files and optional sort fields:

    $ clojure -M:run --sort fields,to,sort input.csv...
    record1
    record2



## Tests

Run tests

    $ clojure -M:test:runner
